Contact Me Securely

The form below allows you to send me a private, secure communication. The message you type will be encrypted before it is sent to that only I can read its contents. This process is 1-way, so please include a means by which I can respond to you. Prefer your command-line to a web form? You can also send me a message from your terminal with the following command (you'll need to import my key first). curl --data-urlencode "message=`gpg -e -a -r E7640EB9`" http://contact.jefflarkin.com/messages After hitting enter, type your message followed by Ctrl-D. You should receive a copy of the encrypted message back from the server.

How it works

This page is based on OpenPGP.JS, a JavaScript implementatipon on the OpenPGP standard. I have embedded my public key into this page. When submitted, your browser will encrypt the message using my public key so that only I can read it. Encrypting it in your browser ensures that no unencrypted data is ever sent, thus only I can ever read what you type. If you would like to implement something similar on your website, I will be providing my client and server code as a reference for how anyone can provide this sort of secure communication.
